Getting Started

Frameworks

Simply Responsive:

  1. Download full framework from here
  2. Include the CSS in the <head> element of your webpage.
  3. Include the JS before the </body> tag of your webpage.

Simply Non-responsive:

  1. Download full framework from here
  2. Include the CSS in the <head> element of your webpage.

Starting Template

Copy the following to use as a starting point for your Simply Responsive project

	<!DOCTYPE html>
	<html lang="en">
		<head>
			<meta charset="utf-8" />
			<meta http-equiv="X-UA-Compatible" content="IE=edge" />
			<meta name="viewport" content="width=device-width, initial-scale=1" />

			<title>Simply Responsive Template</title>

			<!-- Simply Responsive CSS -->
			<link href="css/simplyResponsive.css" rel="stylesheet" />
		</head>
		<body>
			<h1>Hello, we are using Simply Responsive, The Pure JS/CSS Framework!</h1>

			<!-- Optional - Only needed for the full framework -->
			<script src="js/simplyResponsive.js"></script>
		</body>
	</html>
